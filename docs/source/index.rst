Welcome to :red:`mlvern` Documentation
===============================================================

Welcome to the **mlvern** ! A lightweight framework for reproducible machine learning workflows.
It helps you organize datasets, track experiments, version models, and record metrics with clarity and consistency.

This documentation will guide you through the installation process, basic usage, and advanced features of mlvern.
